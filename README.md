# 운친 (運親)

친구의 운동 기록이 나를 움직이게 만드는 소셜 동기부여 웹 서비스

## 프로젝트 정보
- **유형**: 소셜 커뮤니티 / 동기부여 플랫폼
- **타겟 사용자**: 운동 입문자, 꾸준히 못 하는 사람
- **플랫폼**: 웹 (PC 최적화 + 모바일 반응형)

## 핵심 기능
- 30초 안에 완료되는 빠른 운동 기록
- 친구 운동 피드 (좋아요/댓글)
- 다중 그룹 시스템 (기록 1회 → 전체 그룹 자동 공유)
- 친구 챌린지
- 스트릭 & 뱃지 동기부여 시스템

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
