# 운친 — 서비스 기획서

> 친구의 운동 기록이 나를 움직이게 만드는 소셜 동기부여 웹 서비스

---

## 1. 서비스 개요

**운친(運親)**은 운동을 가르치거나 분석하는 서비스가 아니다.
친구들의 운동 기록이 피드에 올라오는 것만으로 "나도 해야겠다"는 동기를 만드는 **소셜 동기부여 플랫폼**이다.

> 핵심 철학: 운동을 가르치지 않는다 — 운동하고 싶게 만든다.

| 항목 | 내용 |
|------|------|
| 서비스명 | 운친 (運親, 운동친구) |
| 플랫폼 | 웹 (PC 최적화 + 모바일 반응형) |
| 타겟 | 운동 입문자, 꾸준히 못 하는 사람 |
| 로그인 | 카카오 / 구글 소셜 로그인 |
| 톤&매너 | 에너지풍 / 짜릿한 (오렌지·레드 계열) |

---

## 2. 사용자 페르소나

**주 타겟: 운동 입문자 / 꾸준히 못 하는 20~30대**
- 운동 자체는 알지만 혼자 하면 금방 그만두게 됨
- SNS를 자주 사용하며 친구 소식에 반응함
- 친구가 올린 운동 인증샷을 보면 자극받아 따라 하는 경향

---

## 3. 온보딩 (3단계)

| 단계 | 내용 | 목적 |
|------|------|------|
| STEP 1 | 카카오 / 구글 로그인 | 빠른 진입, 마찰 최소화 |
| STEP 2 | 닉네임 설정 | 개인 정체성 부여 |
| STEP 3 | 첫 운동 기록 유도 | "오늘 뭐 했어요? 올려봐요!" |

---

## 4. 핵심 기능

### 4-1. 빠른 운동 기록 (목표: 30초 이내)

운동 기록 1건 구성:
- **운동 이름** (자유 텍스트)
- **사진 첨부** (선택)
- **운동 시간** (분 단위)
- **한 줄 코멘트** (선택)

### 4-2. 친구 피드
- 팔로우한 친구의 운동 기록 타임라인
- 좋아요 / 댓글
- 오늘 운동한 친구 수 상단 표시

### 4-3. 그룹 시스템
- 다중 그룹 동시 소속 (고등학교 친구 / 대학교 / 직장 등)
- 기록 1회 업로드 → 전체 그룹 자동 공유
- 그룹별 공유 끄기 설정 가능

**참여 방법:** 초대 링크 / 닉네임 검색 / 카카오 연락처 연동 / QR코드

### 4-4. 챌린지
- 그룹/친구 단위 챌린지 생성
- 기간·목표 설정 (예: 30일 동안 주 3회 운동)
- 진행률 실시간 비교
- 완료 시 뱃지 지급

### 4-5. 스트릭 & 뱃지
- 연속 운동일 스트릭 🔥
- 7일/30일/챌린지 완료 뱃지
- 프로필에 전시

### 4-6. 알림 4종

| 알림 | 예시 |
|------|------|
| 친구 운동 완료 | "홍길동이 오늘 운동을 완료했어요 🔥" |
| 좋아요/댓글 | "홍길동이 내 기록에 응원을 눌렀어요 ❤️" |
| 스트릭 위험 | "오늘 기록 안 하면 스트릭이 끊겨요! 💔" |
| 챌린지 현황 | "챌린지 D-3, 홍길동이 2위예요 🏆" |

---

## 5. 화면 구성 (5개)

| 화면 | 주요 내용 |
|------|----------|
| 🏠 홈 피드 | 친구 운동 타임라인, 그룹 탭 필터, 스트릭 현황 |
| ⚡ 기록하기 | FAB 버튼 → 운동명/시간/사진/코멘트 입력 → 게시 |
| 🏆 챌린지 | 진행 중 챌린지 목록, 진행률 비교, 챌린지 만들기 |
| 🔔 알림 | 4종 알림 목록 |
| 👤 프로필 | 스트릭, 뱃지, 운동 달력, 소속 그룹, 히스토리 |

---

## 6. 핵심 사용자 플로우

```
앱 접속 → FAB 클릭 → 운동명 입력 → (사진/시간/코멘트)
→ 게시 → 전체 그룹 자동 공유 + 스트릭 +1
```
