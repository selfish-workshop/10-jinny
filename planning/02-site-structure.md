# 운친 — 사이트 전체 구조

```
랜딩
 └── 로그인 (카카오/구글)
      └── 온보딩 (닉네임 → 첫 기록)
           └── 메인
                ├── 🏠 홈 피드
                │    ├── 피드 카드 (좋아요/댓글)
                │    └── 그룹 탭 필터
                │
                ├── ⚡ 기록하기 (FAB 모달)
                │    └── 완료 → 전체 그룹 자동 공유
                │
                ├── 🏆 챌린지
                │    ├── 챌린지 목록
                │    ├── 챌린지 상세 (진행률)
                │    └── 챌린지 만들기
                │
                ├── 🔔 알림 (4종)
                │
                └── 👤 내 프로필
                     ├── 스트릭 / 뱃지
                     ├── 운동 달력
                     ├── 소속 그룹 관리
                     │    └── 그룹별 공유 on/off
                     ├── 운동 히스토리
                     └── 설정 (알림/로그아웃/탈퇴)
```

## 그룹 공유 흐름

```
기록 1회 업로드
      │
      ├── 📣 개인 피드 (팔로워 전체)
      ├── 👥 고등학교 친구 그룹
      ├── 👥 대학교 친구 그룹
      └── 👥 직장 그룹 (설정에서 끄기 가능)
```
