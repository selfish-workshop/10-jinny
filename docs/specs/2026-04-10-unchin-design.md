# 운친 (運親) — 서비스 기획서

> 친구의 운동 기록이 나를 움직이게 만드는 소셜 동기부여 웹 서비스
> 작성일: 2026-04-10 | 상태: 기획 확정 → 디자인팀 전달

---

## 1. 서비스 개요

### 정의
**운친(運親)**은 운동을 가르치거나 분석하는 서비스가 아니다.
친구들의 운동 기록이 피드에 올라오는 것만으로 "나도 해야겠다"는 동기를 만드는 **소셜 동기부여 플랫폼**이다.

### 핵심 철학
> 운동을 가르치지 않는다 — 운동하고 싶게 만든다.

### 기본 정보
| 항목 | 내용 |
|------|------|
| 서비스명 | 운친 (運親, 운동친구) |
| 플랫폼 | 웹 (PC 최적화 + 모바일 반응형) |
| 타겟 사용자 | 운동 입문자, 꾸준히 못 하는 사람 |
| 로그인 | 카카오 / 구글 소셜 로그인 |
| 톤&매너 | 에너지풍 / 짜릿한 (오렌지·레드 계열) |

---

## 2. 사용자 페르소나

**주 타겟: 운동 입문자 / 꾸준히 못 하는 20~30대**
- 운동 자체는 알지만 혼자 하면 금방 그만두게 됨
- SNS를 자주 사용하며 친구 소식에 반응함
- "운동 루틴을 배우기보다" 일단 오늘 하루 움직이는 것이 목표
- 친구가 올린 운동 인증샷을 보면 자극받아 따라 하는 경향

---

## 3. 온보딩 (3단계)

| 단계 | 내용 | 목적 |
|------|------|------|
| STEP 1 | 카카오 / 구글 로그인 | 빠른 진입, 마찰 최소화 |
| STEP 2 | 닉네임 설정 | 개인 정체성 부여 |
| STEP 3 | 첫 운동 기록 유도 | "오늘 뭐 했어요? 올려봐요!" |

- 온보딩 중 친구/그룹 찾기는 강제하지 않음 (나중에 자연스럽게 유도)
- 첫 기록 완료 시 환영 뱃지 자동 지급

---

## 4. 핵심 기능

### 4-1. 빠른 운동 기록

**목표: 30초 안에 기록 완료**

운동 기록 1건의 구성:
- **운동 이름** (자유 텍스트) — "오늘 러닝 30분", "헬스장 다녀옴" 등
- **사진 첨부** (선택) — 운동 인증샷
- **운동 시간** — 몇 분 했는지 간단 입력
- **한 줄 코멘트** — "오늘 힘들었지만 해냄 💪" 등

기록 완료 시:
- 가입한 모든 그룹 + 개인 피드에 자동 공유
- 스트릭 +1 카운트
- 챌린지 진행률 자동 반영

### 4-2. 친구 피드

- 내가 팔로우하는 친구들의 운동 기록이 시간순으로 표시
- **좋아요** (응원 이모지) / **댓글** 가능
- 피드 상단에 오늘 운동한 친구 수 표시 ("오늘 3명이 운동했어요 🔥")

### 4-3. 그룹 시스템

**구조:**
- 한 사람이 여러 그룹에 동시 소속 가능 (예: 고등학교 친구 그룹, 대학교 그룹, 직장 그룹)
- 기록을 올리면 가입한 모든 그룹에 자동으로 공유됨 (별도 선택 없음)
- 그룹별 설정에서 해당 그룹으로의 공유를 끌 수 있음

**그룹 참여 방법:**
- 🔗 초대 링크 — 카카오톡/문자로 링크 공유
- 🔍 아이디/닉네임 검색 — 앱 내 직접 검색
- 📱 카카오 연락처 연동 — 연락처 중 운친 사용자 자동 탐색
- 📷 QR코드 스캔 — 오프라인에서 빠른 그룹 참여

**그룹 피드:**
- 홈 피드에서 "전체" / 그룹별 탭 전환 가능
- 그룹 피드는 해당 그룹 멤버만 볼 수 있는 폐쇄 공간

### 4-4. 챌린지

- 그룹 단위 또는 친구 1:1로 챌린지 생성
- 기간과 목표 설정 (예: "30일 동안 주 3회 운동")
- 진행률 실시간 비교 표시
- 챌린지 완료 시 전용 뱃지 지급

### 4-5. 스트릭 & 뱃지

**스트릭:**
- 연속 운동일 카운트 (🔥 n일째)
- 프로필에 크게 표시
- 당일 기록 없으면 자정에 리셋

**뱃지:**
- 7일 스트릭, 30일 스트릭 등 달성 뱃지
- 챌린지 완료 뱃지
- 첫 기록 환영 뱃지
- 프로필에서 보유 뱃지 전시

### 4-6. 알림 (4종)

| 알림 | 예시 문구 |
|------|----------|
| 친구가 운동했을 때 | "홍길동이 오늘 운동을 완료했어요 🔥" |
| 좋아요/댓글 받았을 때 | "홍길동이 내 기록에 응원을 눌렀어요 ❤️" |
| 스트릭 위험 | "오늘 기록 안 하면 스트릭이 끊겨요! 💔" |
| 챌린지 현황 | "챌린지 D-3, 홍길동이 2위예요 🏆" |

## 5. 화면 구성 (5개 화면)

### 🏠 홈 피드
- 친구 운동 타임라인 (최신순)
- 상단 탭: 전체 / 그룹별 필터
- 각 피드 카드: 프로필사진 + 닉네임 + 운동명 + 사진(있으면) + 코멘트 + 시간 + 좋아요/댓글
- 상단 배너: 오늘 운동한 친구 수 + 내 스트릭 현황

### ⚡ 기록하기 (FAB 버튼)
- 플로팅 버튼으로 어디서든 접근
- 운동명 텍스트 입력
- 사진 첨부 (갤러리/카메라)
- 운동 시간 입력 (숫자 + 분)
- 한 줄 코멘트 (선택)
- 게시 버튼 → 완료 애니메이션 + 스트릭 +1 피드백

### 🏆 챌린지
- 진행 중인 챌린지 목록
- 챌린지 생성 (그룹/친구 선택 → 기간/목표 설정)
- 챌린지 상세: 멤버별 진행률 바 + 순위

### 🔔 알림
- 시간순 알림 목록
- 읽은 알림 / 안 읽은 알림 구분

### 👤 내 프로필
- 🔥 스트릭 카운트 (크게 표시)
- 📊 이번 달 운동 달력 (한 눈에 보는 활동 현황)
- 🏅 보유 뱃지 목록
- 내 운동 기록 히스토리 (타임라인)
- 팔로우 / 팔로워 수
- 소속 그룹 목록

## 6. 주요 사용자 흐름

### 메인 플로우: 오늘 운동 기록하기
```
앱 접속 → FAB 버튼 클릭 → 운동명 입력 → (사진/시간/코멘트 선택 입력)
→ 게시 → 전체 그룹 자동 공유 + 스트릭 +1 → 홈 피드에 노출
```

### 그룹 참여 플로우
```
초대 링크 수신 or 검색 → 그룹 미리보기 → 참여 →
자동으로 이 그룹의 피드에 내 기록 공유 시작
```

### 챌린지 참여 플로우
```
챌린지 탭 → 친구 챌린지 생성 → 그룹 또는 친구 초대 →
기간/목표 설정 → 시작 → 운동 기록 시 진행률 자동 반영
```
